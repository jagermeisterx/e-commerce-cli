# 🛒 Ecommerce CLI – Módulo 3

Aplicación de consola desarrollada en **Python 3** que simula el funcionamiento básico de una tienda online. Permite explorar un catálogo de productos, buscar artículos, gestionar un carrito de compras y calcular el total a pagar.

Proyecto desarrollado como entrega del **Módulo 3 – Fundamentos de Python** (Bootcamp AD Academy – Sustantiva).

---

## 📦 Funcionalidades

| Opción | Descripción |
|--------|-------------|
| `1` | Ver todos los productos del catálogo |
| `2` | Buscar producto por nombre o categoría |
| `3` | Agregar producto al carrito (con cantidad) |
| `4` | Ver carrito y total a pagar |
| `5` | Vaciar el carrito |
| `0` | Salir del programa |
